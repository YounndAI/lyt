> A Lyt vault — federated markdown knowledge with AI agent integration.

This vault is part of a Lyt mesh. The canonical bytes live in Git;
derived state (search index, embeddings) lives in `.lyt/indexes/` and is rebuildable.

Lyt edits only the regions it marks — frontmatter and the
`LYT_README_BEGIN`/`LYT_README_END` managed block. Your prose is never touched,
and every change Lyt makes is a plain-text diff in Git you can review. Edit
freely outside the markers.
